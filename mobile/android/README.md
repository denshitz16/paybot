## Troubleshooting
If you encounter issues with "Live Mode":
1. Verify your `MAYA_BUSINESS_MODE` is set to `live` in the backend environment.
2. Ensure your public IP is whitelisted in the Maya Business Manager.
3. Check the Railway logs for any 401/403 errors.
